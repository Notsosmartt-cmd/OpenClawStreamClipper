One-time media hosting setup. Buffer's API can't accept file uploads — each clip must sit at a public HTTPS URL when it's posted. Buffer's docs recommend Cloudinary (free tier, plenty for short clips):
  1. Create a free account at cloudinary.com
  2. On the console home (Dashboard → "Go to API Keys"), copy Cloud name, API Key, and API Secret
  3. Paste them here and Save — they're stored locally in config/buffer_poster.json (gitignored)

Each selected clip becomes its own post on every checked channel. Instagram videos publish as Reels. Caption defaults to the clip title (the “(B)” / “(Short)” marker is stripped) — edit it on the card; hashtags above are appended to every caption at post time. Rate guard: daily caps are checked live (TikTok 25/day, IG 50/day — network caps, same on every Buffer plan); “Post now” allows a small burst per channel and auto-schedules the rest, because TikTok locks the channel for ~24 h if many posts land in rapid succession.
