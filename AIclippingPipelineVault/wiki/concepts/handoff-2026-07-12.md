## 1. Caption overhaul + A/B variants — SHIPPED, PROMOTED DEFAULT-ON

Plan: [[concepts/plan-captions-and-ab-variants-2026-07]] (status: shipped).

**Root causes found (why owner critiques kept recurring):** the owner-visible `title`/`hook`
fields bypassed ALL semantic verification (`_CREATIVE_FIELDS`, overlap 0.0, no judge) while the
unseen `description` got the full cascade; the vision prompt's transcript was truncated `[:500]`
(~25 s) so long clips were titled from a fraction of their words; the Phase-7.2 caption-voice
profile was disabled AND poisoned by EasyOCR garble.

**Shipped default-ON** (kill switches `CLIP_CAPTION_JUDGE/LINT/VOICE=0`): full-window transcript
`[:4000]`; `grounding.caption_judge` (fidelity+voice, pinned to the loaded Stage-6 model, drives
the existing regenerate-once path); voice-contract prompt; `caption_lint.py` deterministic
AI-tell linter (self-test fixtures = the owner's own critiqued titles). **Live-proof:** the
judge nulled exactly ONE title in the verification run — fidelity 2/10 with a dead-on rationale.

**A/B (owner decisions: 2 variants, no hashtags, varied AV):** variant B = full independent
`profile_render` with alternate-angle hook + `--seed-offset 1` (different SFX/effect draws;
beats stay anchored); post kit `.post.json` per clip (TikTok/IG/YT copy, NO hashtags) →
**`clips/post_kits/`** (owner: keep the clips root video-only). **Owner review of the first
A/B run: 9/9 GOOD → everything promoted default-on** (`CLIP_AB_VARIANTS=2`, `CLIP_POST_KIT`,
`CLIP_STYLE_PROFILES` incl. `DEFAULT_ORIGINALITY`). 9 labels filed → owner pool 35 (33/2).

## 3. "Streamers Update" news compilation — v1.5 SHIPPED, gates open

Plan: [[concepts/plan-news-compilation-2026-07]]. Born FROM the attribute cards (the
StreamerUpdate reference set decoded the format; first real consumer of loop B).

**Architecture:** stories are payoff sub-cuts of FINISHED clips (captions/SFX inherited — the
companion-short insight cross-VOD); trace×effects×disk join (normalized PREFIX title match —
filenames are `[:50]`-truncated); news-weighted selection (controversial ×1.35, storytime ×1.2,

`CLIP_AB_VARIANTS` (B = stories from `(B)` clip renders + rotated SFX). Outputs verified: A+B
44.9 s each.

`News Compile (N)` button on the multi-select, and the `chk-news-after` toggle
(`CLIP_NEWS_AFTER`) that ends a normal run with one compilation from the freshly clipped VODs.

the default engine** (Apache-2.0 ONNX, natural prosody, 8.3 s audio in 3.0 s CPU; engine chain

## Tested-and-ABANDONED routes (and why) — keep these from being re-tried blindly

| Route | Verdict | Why |
|---|---|---|
| EasyOCR as the caption-LANGUAGE source | **Abandoned** (kept only for density stats) | Garble/watermarks poisoned voice-bank v1 ("yoU ever heard of george TkTok"); the VLM reads on-screen text from frames instead — proven clean on the exact failure string |
| piper ryan-high as the news anchor | **Demoted to fallback** | Owner: "very noticeably ai with its tonality and cadence" → Kokoro-82M default; piper stays in the failure-soft chain |
| Shared-master + drawtext-only pass for A/B variants | **Invalidated** | Owner chose varied SOUND+VISUAL per variant — a shared master can't carry per-variant AV; B is a full independent render |
| WebFetch on myinstants pages | **Blocked (403)** | Cloudflare bot-blocks the fetcher; curl with a browser UA on the CDN paths works |
| Root-only `clips/*.mp4` globs | **Abandoned** | The owner reorganizes finished clips into subfolders + moves posted ones OFF-repo; all clip lookups are now recursive root-first with clear missing-source logs |
| Bare-list SFX manifests | **Wrong shape** | `sfx_inject._candidates_for_kind` requires `{"entries": […]}` |
| Raw CLAP counts as the reference SFX measure | **Flagged, pending fix** | Over-fires on speech (28× "bruh" on one clip) → reference `31/30s` inflated; same-instrument fix (dedupe windows + drop speech-like labels) = top R3 refinement before trusting magnitudes |
| Equality title matching (effects_log ↔ filenames) | **Abandoned** | Filenames are the title `[:50]`-truncated AND sanitized → normalized PREFIX match everywhere |
| Old Clip Forensics tab | **Deleted** | Single-clip decompose subsumed by the Reference Lab's corpus Decompose button |
| `category:"irl"` coarse card taxonomy | **Replaced** | One bucket hid formats distinct enough to spawn a pipeline mode → 10 finer categories |
