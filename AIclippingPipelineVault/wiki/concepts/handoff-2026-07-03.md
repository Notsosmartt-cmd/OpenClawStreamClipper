## The harness — how to run any gated phase (unattended)
```
# system python:
python scripts/research/phase_runner.py launch  --vod NAME --force --profile validation --label RUN --phase N
# wait (STALL-AWARE — catches hangs; run in a background shell so its notification wakes you):
python scripts/research/phase_runner.py wait     --run RUN --stall 600 --timeout 5400
python scripts/research/phase_runner.py status   --run RUN
python scripts/research/phase_runner.py evaluate --run RUN --phase N --forensics [--baseline RUN0]
python scripts/research/phase_runner.py state    --advance PASS|FAIL --phase N
```
`launch` is fully detached (survives the agent sandbox). A run ≈ 30–45 min; launch it, start the watcher, build the NEXT phase's code while it runs, evaluate when woken.
